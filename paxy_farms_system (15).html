🔥 Connect Paxy Farms to Firebase

One-time setup — data syncs live across all devices

Step 1 — Create a free Firebase projectGo to console.firebase.google.com → Add Project → Name it Paxy Farms → Disable Google Analytics → Create Project
Step 2 — Enable Firestore DatabaseLeft menu → Firestore Database → Create Database → Start in test mode → Region: europe-west1 (closest to Nigeria) → Enable
Step 3 — Get your config keysClick ⚙️ gear → Project Settings → scroll to Your apps → click </> Web → Register app as Paxy Farms → copy the config values below
✅ Connected! Loading your farm data...

Config is saved in your browser — you only enter this once per device. All farm data lives in Firebase and is shared across every device in real time.
